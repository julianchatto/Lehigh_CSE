In this assignment we will explore the C GCC compilation pipeline. I've defined a simple C program in the project directory. 2. The C frontend - cc
3. The assembler - as

- `.h` - C Header file

- `.i` - C preprocesor output
- `.s` - Assembly

- `.a` - Object Archive
